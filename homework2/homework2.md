Native app, Web app and Hybrid app
====
<br>

## 一.介绍

<br>

### 1.app的分类


app大致可以分为：
* native app(本地app)
* web app(网页app)
* hybrid app(混合app)

### 2.三类app的定义
#### 2.1 Native app
>基于智能手机本地操作系统如iOS、Android、WP并使用原生程式编写运行的第三方应用程序，一般开发的语言为Java、C++等。           ——百度百科
<br>

#### 2.2 Web app
>基于web的系统和应用，运行于网络和浏览器之上，目前多采用h5标准开发。——百度百科
<br>

#### 2.3 Hybrid app
>Hybrid App（混合模式移动应用）是指介于web-app、native-app这两者之间的app，兼具“Native App良好用户交互体验的优势”和“Web App跨平台开发的优势”。——百度百科
